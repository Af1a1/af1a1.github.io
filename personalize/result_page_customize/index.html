    Setting up Result page from dashboard¶

    Edit recommendation content¶

    To edit recommendation content you need to follow these steps:

    1. Access Visual Quiz Builder app from store dashboard

    2. In Visual Quiz Builder app, open your quiz

    3. In the dashboard, scroll down until you find Result page section and press on it

    4. In the right sidebar disable toggle button find Enable custom recommendation URL toggle button.
